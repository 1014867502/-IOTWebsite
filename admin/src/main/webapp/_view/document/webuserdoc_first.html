设备管理平台介绍

设备管理平台主要管理的是普适型监测设备，分为网页版和APP版。此文主要说明网页版设备管理平台的使用方法。
用法

设备管理平台网址：http://8.134.94.147:8080/ 在浏览器输入http://8.134.94.147:8080/进入图1-1登录界面，输入用户名、密码，点击【登入】进入设备管理平台界面。

图1-1

设备管理平台如图1-2所示，分为左右两部分，左侧导航，右侧根据导航显示内容，右上角是用户信息，可以重置密码、退出当前登录账号。

图1-2
注意：不同的浏览器显示的效果可能会有稍许差别，建议使用谷歌浏览器或是IE8以上版本。当信息显示不全时，请定期清空浏览器缓存。
