用户管理

用户管理页面可对账号进行管理
用法

用户管理如图5-1所示，在用户列表中可以看到管理员账号所属公司下的所有用户信息。

可对表格中的设备进行搜索、批量删除、批量重置密码操作。搜索设备的关键词是设备SN号。

图5-1

点击【添加用户】如图5-2所示，需要填写用户基础信息和用户权限。基础信息包括账号名（唯一且不可改变）、昵称、登录密码、所属部门。用户权限用户类型（有普通用户、供销商管理员、安装测试人员）、可查看项目、网页功能权限、APP功能权限、账号时限、修改设备功能权限和安装功能权限。

图5-2

选择一条用户信息，点击【编辑】如图5-3所示，可以修改用户基础信息和用户权限。基础信息包括昵称、登录密码、所属部门。用户权限角色类型（有普通用户、供销商管理员、安装测试人员）、网页功能权限、APP功能权限、修改设备功能权限和安装功能权限。

图5-3

选择一条用户信息，点击【删除】可以删除该用户；点击【重置密码】可以将用户密码改为123456。
